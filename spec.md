# Specification

## Summary
**Goal:** Revert the entire project (frontend and backend) to its exact Version 9 state.

**Planned changes:**
- Restore all frontend and backend files to their Version 9 state
- Remove all changes introduced after Version 9, including Instagram/social feed cleanup, social media manager, dynamic socialPlatforms CMS extension, and social share buttons in Lightbox

**User-visible outcome:** The application runs exactly as it did at Version 9, with no features or modifications from subsequent versions present.
