# Specification

## Summary
**Goal:** Fix the footer bottom bar so that the "Admin Sign In" link and all other text in that section are clearly legible against the background.

**Planned changes:**
- Update the Footer component's bottom section to explicitly set a background color that provides sufficient contrast for all text elements.
- Ensure the "Admin Sign In" link, visitor counter labels ("Total Visits", "Unique Visitors"), and copyright text all use a contrasting color (e.g., warm beige #F5F0E8 or white #FFFFFF on a dark background, or charcoal #2C2C2C on a light background).
- Apply consistent contrast-compliant color tokens from the existing design system across all viewport sizes.

**User-visible outcome:** All text in the footer bottom bar — including the "Admin Sign In" link, visitor counters, and copyright — is fully readable without any user interaction, on all screen sizes.
